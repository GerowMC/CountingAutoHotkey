# CountingAutoHotkey
It asks you for a starting number, number of loops, and a delay in seconds. It starts by typing the starting number, and then types that number plus one, and that how many ever times you mentioned in the number of loops. It will type and press enter every amount of time mentioned in the delay. Then you just do Ctrl+J, follow the instructions, and it will start counting!
For anybody intending, PLEASE DO NOT use this script for cheating reasons. Thank you.
